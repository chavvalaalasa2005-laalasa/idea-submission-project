# BitZen - AI-Based Wind Turbine Blade Inspection

## Project Title
Autonomous AI-Driven Visual Inspection and Predictive Maintenance System for Wind Turbine Blades

## Team
- Team Name: BitZen
- Team Leader: Chavva Laalasa
- Hackathon: AI Bharat
- Domain: AI for Communities, Access & Public Impact

## Overview
This project provides an autonomous AI-powered drone-based system for inspecting wind turbine blades. The system uses computer vision and predictive analytics to detect blade damage, reduce manual inspection risks, and improve maintenance efficiency.

## Key Features
- Drone-based autonomous inspection
- AI-based blade damage detection
- Anomaly detection and predictive maintenance
- Web dashboard and mobile application
- AR-guided repair support

## Repository Contents
- requirements.md — Project requirements and specifications
- design.md — System architecture and design

## Impact
The system improves safety, reduces downtime, lowers inspection costs, and supports India’s renewable energy goals.
